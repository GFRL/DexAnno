How to Use

1.Select Grasp Type (Left-click in the right panel)
Current type: None

2.Pick Grasp Point (Right-click on the object)
Point coordinates: None

3.Generate Grasp / Skip Object (Left-click at the bottom right)

4.Annotate Grasp as Success / Failure (Left-click at the bottom right)

Common Issue

1.Slow Processing
Backend runs on a single Linux PC (1 GPU + 2 CPUs)
Too many users online (current: -)(May lag if >10 users)
Poor network connection

2.Failed Grasp Generation
Object is too large or too small
Selected grasp type is unsuitable
